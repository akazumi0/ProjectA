## üì• Download Workflow

### Step 1: Download Kenney Packs (Essential Graphics)

```bash
cd /home/user/ProjectA/Assets/Graphics/kenney/

# Download Space Shooter Redux
wget -O space-shooter-redux.zip "https://kenney.nl/media/pages/assets/space-shooter-redux/b31285e83d-1677669442/kenney_space-shooter-redux.zip"

# Download Particle Pack
wget -O particle-pack.zip "https://kenney.nl/media/pages/assets/particle-pack/1dd3d4cbe2-1677578741/kenney_particle-pack.zip"

# Extract
unzip space-shooter-redux.zip -d space-shooter-redux/
unzip particle-pack.zip -d particle-pack/

# Clean up
rm space-shooter-redux.zip particle-pack.zip
```

### Step 2: Download Planet Sprites

```bash
cd /home/user/ProjectA/Assets/Graphics/planets/

# Download planets (requires 7zip)
wget "https://opengameart.org/sites/default/files/planets.7z"

# Extract (install 7zip first if needed: sudo apt-get install p7zip-full)
7z x planets.7z

# Clean up
rm planets.7z
```

### Step 3: Download Sound Effects (99Sounds)

1. Visit: https://99sounds.org/sci-fi-sounds/
2. Enter email to receive download link
3. Download ZIP file
4. Extract to: `/home/user/ProjectA/Assets/Sounds/99sounds/`

### Step 4: Download Music Tracks

**Manual Downloads (via browser):**
1. Visit Chosic, Incompetech, or Pixabay
2. Search for space/ambient tracks
3. Download 3-5 tracks
4. Save to: `/home/user/ProjectA/Assets/Music/ambient/`

**Keep Track of Attribution:**
- Create `music-credits.txt` with track names and artist info

## ‚ö° Integration Quick Start

### After Downloading Graphics:

1. **Extract all ZIP files** to their respective folders
2. **Organize sprites** by category (fragments, particles, UI, etc.)
3. **Create sprite sheets** if needed (use tools like TexturePacker or manual CSS sprites)
4. **Update game code** to load images instead of procedural rendering:

```javascript
// Example: Load fragment sprite
const fragmentImage = new Image();
fragmentImage.src = '/Assets/Graphics/fragments/star-01.png';

// Use in canvas rendering
ctx.drawImage(fragmentImage, x, y, width, height);
```

### After Downloading Music:

1. **Place MP3 files** in `/Assets/Music/ambient/`
2. **Create playlist** in `audio.js`:

```javascript
const musicTracks = {
  ambient: [
    { name: 'Space Ambient 01', file: '/Assets/Music/ambient/track01.mp3' },
    { name: 'Space Ambient 02', file: '/Assets/Music/ambient/track02.mp3' }
  ]
};
```

3. **Add attribution** to credits file

### After Downloading Sounds:

1. **Organize by category** (ui/, gameplay/, events/)
2. **Integrate into audio system**:

```javascript
const soundEffects = {
  click: new Audio('/Assets/Sounds/ui/click-01.wav'),
  capture: new Audio('/Assets/Sounds/gameplay/capture-01.wav'),
  achievement: new Audio('/Assets/Sounds/events/achievement-unlock.wav')
};
```

## üÜò Troubleshooting

### "wget: command not found"
```bash
# Install wget (Ubuntu/Debian)
sudo apt-get install wget

# Or use curl instead
curl -L -O "https://kenney.nl/media/pages/assets/space-shooter-redux/b31285e83d-1677669442/kenney_space-shooter-redux.zip"
```

### "7z: command not found"
```bash
# Install 7zip
sudo apt-get install p7zip-full
```

### "unzip: command not found"
```bash
# Install unzip
sudo apt-get install unzip
```

### Large file downloads timing out
- Download via browser instead
- Use download manager for large files
- Try alternative sources (OpenGameArt vs Kenney.nl)
